# Preview Backup Layer

Preview Backup Layer is a Glyphs reporter plugin.

## Installation

Current manual install:

1. Copy `PreviewBackupLayer.glyphsReporter` to:

```text
~/Library/Application Support/Glyphs 3/Plugins/
```

2. Restart Glyphs.
3. Enable/open the reporter from Glyphs as needed.

Package-manager install instructions will be added once the plugin is listed in Glyphs packages.
